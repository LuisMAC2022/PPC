PPC

Programación Paralela y Concurrente es el arte de dividir y distribuir una tarea con la intención de realizarla en menos tiempo entre múltiples computadoras.

Este proyecto busca conectar computadoras personales y así montar una red que sirva de lienzo para practicar este tipo de programación.
